# CENTINELA - CRISIS FOCUS MODE SPEC

Generated: 2026-05-26

## Purpose

Crisis Focus Mode reduces cognitive load during critical security pressure.

## Activation Conditions

- CRITICAL threat active.
- Governance compromised.
- Source trace missing during release.
- Repeated tool abuse.
- Data leakage risk.
- Ecosystem propagation risk.

## Display Rules

Show only:

- critical threat,
- affected systems,
- containment action,
- freeze state,
- rollback status,
- CEO approval needed,
- last trusted timestamp.

Hide or collapse:

- decorative charts,
- non-critical historical lists,
- low-priority alerts,
- redundant metrics,
- noisy technical detail.

## Required Actions

- Freeze release.
- Preserve logs.
- Preserve previous stable state.
- Request CEO decision if irreversible action is required.

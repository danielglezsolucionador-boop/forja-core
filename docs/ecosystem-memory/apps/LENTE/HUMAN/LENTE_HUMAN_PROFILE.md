# LENTE — HUMAN CABIN PROFILE

## PROPÓSITO HUMANO

Lente NO debe sentirse como:
- una app IA genérica
- un editor barato de reels
- una fábrica automática de videos
- una herramienta visual saturada
- un dashboard lleno de ruido

Debe sentirse como:

un estudio creativo cinematográfico premium diseñado para transformar ideas, narrativa y estrategia en contenido visual elegante, poderoso y reputacionalmente sólido.

━━━━━━━━━━━━━━━━━━━

# OBJETIVO HUMANO

Permitir que:

- CEO
- Cerebro
- Marketing
- Pluma

puedan:
- crear
- visualizar
- revisar
- transformar
- organizar
- descargar
- aprobar

contenido visual premium de manera natural, intuitiva y elegante.

━━━━━━━━━━━━━━━━━━━

# ALERTAS

Lente debe alertar SOLO cuando:

- un render terminó
- una campaña terminó
- un branding terminó
- existe oportunidad visual importante
- existe riesgo reputacional visual
- existe degradación visual importante

NO:
spam operacional.

El sistema debe evitar:

- exceso de efectos
- exceso de colores
- exceso de animaciones
- exceso de paneles
- exceso de ruido visual
- interfaces agresivas

Priorizar:

- respiración visual
- claridad
- foco creativo
- navegación intuitiva
- elegancia
